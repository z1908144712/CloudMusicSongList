# 网易云音乐歌单导出工具

通过解析网易云音乐PC版本地歌单数据库进行实现

思路来源: [网易云音乐怎么导出歌单？ - Timothy Flankl的回答 - 知乎](https://www.zhihu.com/question/31816805/answer/282019014)

# 使用

在输入webdb.dat数据库路径后，需要依次选择uid和pid，uid是用户id，pid是歌单id，tid是歌曲id。
